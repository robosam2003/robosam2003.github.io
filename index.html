Projects

Maxwell

Maxwell is an open source, 3-phase motor controller, specifically made for robot actuators.

Specifications:

  • Up to 60V input
  • 70A continuous, 150A peak
  • PWM, CAN bus and USB control
  • Built in absolute magnetic encoder
  • Support for external SPI / hall effect encoders

Real Steel

Real Steel is a Heavyweight BattleBot, designed to shove and push it's way to victory!

Features:

  • 10kW Four-wheel drive - Has been known to pull fully loaded vans!
  • Rear go-cart wheels for maximum shove
  • Hardox-450 armour (used on tanks and military vehicles)
  • 10mph top speed

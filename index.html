Projects

Maxwell

Maxwell is an open source, 3-phase motor controller, specifically made for robot actuators.

Specifications:

  • Up to 60V input
  • 70A continuous, 150A peak
  • PWM, CAN bus and USB control
  • Built in absolute magnetic encoder
  • Support for external SPI / hall effect encoders

Real Steel

Real Steel is a Heavyweight BattleBot, designed to shove and push it's way to victory!

Features:

  • 10kW Four-wheel drive - Has been known to pull fully loaded vans!
  • Rear go-cart wheels for maximum shove
  • Hardox-450 armour (used on tanks and military vehicles)
  • 10mph top speed

Hound

Hound is a featherweight class combat robot - He's just a bit of fun.

  • Four-wheel drive - old drill motors and gearboxes.
  • Steel armour
  • 20mph top speed
  • Terrible in the snow. :/

Knightcrawler Quadruped

A four legged robot that has taken on many forms, including Batman's "Knightcrawler".

  • Robot Dog configuration worked best
  • 12X high torque servo drives, running off a servo driver board @7V
  • PS3 controller control
  • Strafing, walking and turning gaits

Autonomous Factory navigation

Designed as a test platform for autonomous navigation techniques, this robot uses LIDAR mapping to navigate to waypoints safely.

  • Four-wheel drive - Stepper motors
  • Single point Lidar on a servo motor
